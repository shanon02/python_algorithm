### **문제**  

정수 배열 num_list가 주어질 때,   
가장 첫 번째 원소를 1번 원소라고 할 때, 홀수 번째 원소들의 합과 짝수 번째 원소들의 합 중 큰 값을 반환하세요.   
두 값이 같을 경우 그 값을 반환합니다.

### **예시**  

**Input 1 / Output 1**:  
num_list = [4, 2, 6, 1, 7, 6]  
<br/>
result = 17
<hr/>

**Input 2 / Output 2**:  
num_list = [-1, 2, 5, 6, 3]  
<br/>
result = 8
