### **문제**  

한 자리 정수로 이루어진 문자열 num_str이 주어질 때,  
각 자리수의 합을 return하도록 solution 함수를 완성해주세요.

### **예시**  

**Input 1 / Output 1**:  
num_str = "123456789"  
<br/>
result = 45
<hr/>

**Input 2 / Output 2**:  
num_str = "1000000"
<br/>
result = 1
